## List Conversations

```
GET /conversations
```

Returns a paginated list of conversations for the specified user, sorted by update time by default.

### Query Parameters

<ParamField query="user" type="string" required>
  The end user identifier.
</ParamField>

<ParamField query="last_id" type="string">
  The ID of the last conversation on the current page. Used for cursor-based pagination.
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Number of conversations to return. Must be between `1` and `100`.
</ParamField>

<ParamField query="sort_by" type="string" default="-updated_at">
  Sort order. One of: `created_at`, `-created_at`, `updated_at`, `-updated_at`. A leading `-` indicates descending order.
</ParamField>

### Response

<ResponseField name="limit" type="integer">
  Number of conversations returned.
</ResponseField>

<ResponseField name="has_more" type="boolean">
  Whether there are more conversations to fetch.
</ResponseField>

<ResponseField name="data" type="array">
  List of conversation objects.

  <Expandable title="Conversation object">
    <ResponseField name="id" type="string">Conversation ID.</ResponseField>
    <ResponseField name="name" type="string">Conversation title.</ResponseField>
    <ResponseField name="inputs" type="object">Input variables for the conversation.</ResponseField>
    <ResponseField name="status" type="string">Conversation status.</ResponseField>
    <ResponseField name="introduction" type="string">App opening statement shown at the start.</ResponseField>
    <ResponseField name="created_at" type="integer">Unix timestamp of creation.</ResponseField>
    <ResponseField name="updated_at" type="integer">Unix timestamp of last update.</ResponseField>
  </Expandable>
</ResponseField>

### Example

```bash
curl 'https://api.dify.ai/v1/conversations?user=user-abc123&limit=20' \
  -H 'Authorization: Bearer {api_key}'
```

## Delete Conversation

```
DELETE /conversations/{conversation_id}
```

### Path Parameters

<ParamField path="conversation_id" type="string" required>
  The UUID of the conversation to delete.
</ParamField>

### Request Body

<ParamField body="user" type="string" required>
  The end user identifier.
</ParamField>

### Response

Returns `204 No Content` on success.

### Example

```bash
curl -X DELETE https://api.dify.ai/v1/conversations/3fa85f64-5717-4562-b3fc-2c963f66afa6 \
  -H 'Authorization: Bearer {api_key}' \
  -H 'Content-Type: application/json' \
  -d '{"user": "user-abc123"}'
```

## Rename a Conversation

```
POST /conversations/{conversation_id}/name
```

Sets a custom title for a conversation, or auto-generates one from the conversation history.

### Path Parameters

<ParamField path="conversation_id" type="string" required>
  The UUID of the conversation to rename.
</ParamField>

### Request Body

<ParamField body="name" type="string">
  New title for the conversation. Required when `auto_generate` is `false`.
</ParamField>

<ParamField body="auto_generate" type="boolean" default="false">
  When `true`, the API generates a name automatically from the conversation content. `name` is ignored.
</ParamField>

<ParamField body="user" type="string" required>
  The end user identifier.
</ParamField>

### Response

Returns the updated conversation object.

<ResponseField name="id" type="string">Conversation ID.</ResponseField>
<ResponseField name="name" type="string">Updated conversation title.</ResponseField>
<ResponseField name="inputs" type="object">Input variables for the conversation.</ResponseField>
<ResponseField name="status" type="string">Conversation status.</ResponseField>
<ResponseField name="created_at" type="integer">Unix timestamp of creation.</ResponseField>
<ResponseField name="updated_at" type="integer">Unix timestamp of last update.</ResponseField>

### Example

```bash
curl -X POST https://api.dify.ai/v1/conversations/3fa85f64-5717-4562-b3fc-2c963f66afa6/name \
  -H 'Authorization: Bearer {api_key}' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "My First Chat",
    "auto_generate": false,
    "user": "user-abc123"
  }'
```

```json Response
{
  "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "name": "My First Chat",
  "inputs": {},
  "status": "normal",
  "created_at": 1705395200,
  "updated_at": 1705395400
}
```
